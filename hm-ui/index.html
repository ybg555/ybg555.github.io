HM UI

基于 Element UI 2.x 的管理后台组件库

快速上手 →

快速开发

参照指引文档和示例代码，可以帮助您快速的实现业务功能。

Vue驱动

享受 Vue + webpack 的开发体验，可以在 Markdown 中使用 Vue 组件，又可以使用 Vue 来开发自定义主题。

高性能

VuePress 为每个页面预渲染生成静态的 HTML，同时在页面被加载的时候，将作为 SPA 运行。

# 像数 1, 2, 3 一样容易

# 安装
npm i -S hm-ui

# 使用
import Vue from 'vue';
import HmUi from 'hm-ui';

Vue.use(HmUi);

TIP

请确保你的 Node.js 版本 >= 6
