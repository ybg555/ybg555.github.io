# thousands
数字千分位格式化。

## 示例图
123456.30231 => 123,456.30

## 代码演示
依赖
* Element 2.x

基本用法
```html
<hm-thousands num="123456.30231"></hm-thousands>
```

## 参数说明
| 参数 | 说明 | 类型|  可选值 | 默认值 |
|-----|:----------|:------|:------------|:--------|
| num | 数字 |  Number or String | － | - |
| decimal | 小数位数 |   Number | － | 2 |
| convert | 是否开启千分位转换 |  Boolean | － | true |
