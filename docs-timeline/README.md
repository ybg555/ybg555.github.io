### 像数 1, 2, 3 一样容易

``` bash
# 安装
npm i -S hm-ui

# 使用
import Vue from 'vue';
import HmUi from 'hm-ui';

Vue.use(HmUi);
```

::: tip
请确保你的 Node.js 版本 >= 6
:::
